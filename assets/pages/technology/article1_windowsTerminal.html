When you start using Linux and following Linux-based websites and forums, you’ll often come across terms like GUI, CLI and sometimes TUI This chapter of Linux Jargon Buster briefly explains these terms so that you, as a (new) Linux user, can understand the context better when these acronyms are used To be honest, the terms GUI, CLI and TUI are not exclusive to Linux. These are generic computing terms that you’ll find used in non-Linux discussions, as well.
